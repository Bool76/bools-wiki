# Block Elements

Summary - One possible value for the "display" attribute for page element's is BLOCK

```
display: block
```

The following html elements by default have a display value of "block":

- p
- h1, h2, h3, h4, h5, h6
- ol, ul
- pre
- address
- blockquote
- dl
- div
- fieldset
- form
- hr
- noscript
- table
